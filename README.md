# JSON 格式化工具

一个基于 Next.js 15.3 的现代化 JSON 格式化工具，使用 TailwindCSS 和 Shadcn/ui 构建。

## 功能
- 📝 JSON 字符串格式化
- 🎨 美观的用户界面
- 📋 一键复制格式化结果
- 📱 响应式设计
- ❌ 友好的错误提示

## 技术栈
- **Next.js 15.3** - React 框架
- **TailwindCSS** - CSS 框架
- **Shadcn/ui** - UI 组件库
- **TypeScript** - 编程语言

## 开发

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

## 部署
项目可以部署到 Vercel、Netlify 或其他支持 Next.js 的平台。
